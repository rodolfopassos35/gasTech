# 🔧 gasTech Serviços

Site institucional da **gasTech Serviços**, empresa especializada em soluções de gás encanado para residências, comércios e indústrias. O projeto foi desenvolvido com foco em performance, responsividade e otimização para mecanismos de busca (SEO).

🌐 Acesse o site: [gastechservicos.com.br](https://gastechservicos.com.br)

---

## 📌 Sobre o projeto

Este site tem como objetivo apresentar os serviços da gasTech, facilitar o contato com clientes e reforçar a presença digital da empresa. O layout é limpo, direto e adaptado para dispositivos móveis.

---

## 🛠️ Tecnologias utilizadas

- **HTML5** – Estrutura semântica e acessível
- **CSS3** – Estilização moderna e responsiva
- **JavaScript** – Funcionalidades interativas
- **Vercel** – Deploy rápido e confiável
